About

The e-Recruit system is a recruitment automation system that aims to simplify the cumbersome process encountered when applying for various opportunities. This system aims to efficiently smoothen the recruitment process as user friendly as possible while maintaining integrity. This is a system intended for both recruiters and applicants wherein recruiters can upload different types of vacancies (opportunities) such as employment vacancies, bursaries, and leanerships then let the system handle most of the selection processes. The system also includes notification functionalities so that users are kept up-to-date with relevant information without having to log in onto the system.
